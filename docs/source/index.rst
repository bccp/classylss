classylss
=========

|

*classylss* is a lightweight Python binding of the CMB Boltzmann code,
`CLASS <http://class-code.net>`_, with an emphasis on the routines that are
important for large-scale structure calculations. The main modules of the
CLASS code are exposed to the user via a Cython wrapper.

classylss is supported for Linux and macOS machines and Python versions 2.7, 3.5 and 3.6.
